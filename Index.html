    Créer un dépôt sur Github

    Depuis https://github.com/NomUtilisateur?tab=repositories

    • cliquer sur l'icône "New"
    • puis "Create Repository"
    • sur la nouvelle page, choisir la rubrique "create a new repository on the command line"

    Créer un dépôt local

    echo "# NomDépot" >> README.md< création du fichier de description README au format MarkDown
    git init Création du dépot (Repository)
    git add README.md ajout du fichier README.md pour la prochaine sauvegarde
    git commit -m "first commit" Première sauvegarde avec un commentaire

    Synchroniser

    git push origin master envoi de la sauvegarde sur le serveur distant
